Dataset Description

This visualization is based on data from linuxcounter.net, a site that collects informations about the diffusion of linux machines in the world. After data preparation, each machine is characterized to four attributes: Distribution, Class, Architecture and Number of Cores. The chart allows to explore the data in a hierarchical way where each attribute represents a node.

Guide

The chart is made of a group of bubbles arranged clockwise based on their size. A bubble consists in a set of machines with properties in common and each attribute represents a specific level of the hierarchy as shown below:

The first way to interact with the graph is to click on a bubble to Zoom inside it. This allows to select machines of a specific hierarchy level. To help the exploration is possible to redraw only a partition of bubbels by selecting Ordered Selection in the Settings panel. Clicking on the radio button the zoom function is disabled and selecting a bubble, all those with size less or equal than selected one, will be highlighted. The aim is to focus the attention on bubbles too difficult to select because characterized of few elements. The Main Chart button will be enabled after the bubbles having been redrawd. Info Nodes shows informations about nodes when mouse goes over it.
