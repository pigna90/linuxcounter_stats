Dataset Description

This view represents the data in linuxcounter.net, a site that stores stats about the spread of linux devices in the world. The data, after the initial processing, contain the set of linux machines characterized by four attributes: Distribution,Class,Architecture and Number of Cores. The chart allows exploring the data in a hierarchical way in which each attribute represents a node.

Guide

The chart is composed of groups of bubbles arranged clockwise to their size. Every bubble represents a set of machines with properties in common and each attribute matches a specific level of the hierarchy as shown below:

The interaction with the graph is possible through clicking on a bubble to zoom on it. This allows to select a set of machines of a specific hierarchy level. It's also possible to reshape only a subset of bubbles through the Ordered Selection option in the Setting panel, a click on the radio button results in the disabling of the zoom function and at this point the selection of a bubble causes the highlighting of the bubbles with size less or equal than the selected one's. This function helps the selection of the smallest bubbles, those that have few elements. The Main Chart button will be enabled only after the bubble will be reshaped. When the mouse override a node the Info Nodes shows information about that node.
